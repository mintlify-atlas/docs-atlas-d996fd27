## Introduction to PHP Functions

Functions are reusable blocks of code that perform specific tasks. They help organize your code, make it more maintainable, and reduce repetition. This collection of exercises will help you master PHP functions through real-world problems.

### Function Fundamentals

<Note>
A function in PHP is defined using the `function` keyword, followed by a name, parentheses for parameters, and curly braces containing the code to execute.
</Note>

**Key Concepts:**

- **Parameters**: Input values passed to a function
- **Return Values**: Output that a function sends back to the caller
- **Scope**: Variables inside functions are local by default
- **Reusability**: Write once, use multiple times

### Basic Function Syntax

```php
function functionName($parameter1, $parameter2) {
    // Function body
    $result = $parameter1 + $parameter2;
    return $result;
}

// Calling the function
$sum = functionName(5, 10);
echo $sum; // Outputs: 15
```
