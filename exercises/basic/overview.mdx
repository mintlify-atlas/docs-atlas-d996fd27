# Basic Exercises Collection

Welcome to the Basic Exercises collection! This comprehensive set of 35 programming exercises will help you build a solid foundation in PHP programming. Each exercise focuses on essential programming concepts and real-world problem-solving techniques.

## Learning Path

This collection is designed to take you from fundamental concepts to more complex programming patterns. The exercises are organized into three main categories:

1. **Conditionals** - Learn decision-making logic and control flow
2. **Loops** - Master iteration patterns and repetitive operations
3. **Arrays** - Understand data structures and manipulation techniques

<Note>
  All exercises include working code examples extracted from real PHP files. You can use these as references or starting points for your own implementations.
</Note>

## Topics Covered

Throughout these 35 exercises, you'll explore:

- **Control Structures**: if/else statements, switch cases, and conditional logic
- **Mathematical Operations**: calculations, conversions, and formulas
- **String Manipulation**: concatenation, searching, and pattern matching
- **Loop Patterns**: for loops, while loops, and iteration strategies
- **Array Operations**: creation, manipulation, filtering, and sorting
- **Functions**: creating reusable code blocks and parameter handling
- **Validation**: data verification and error checking
- **Algorithms**: prime numbers, factorials, Fibonacci sequences, and palindromes
