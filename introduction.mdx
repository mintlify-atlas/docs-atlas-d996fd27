## Overview

This exercise collection is designed for **Programming 2** students to build practical PHP development skills through hands-on coding exercises. The project provides a progressive learning path from basic syntax to full CRUD applications with MySQL databases.

<Note>
All exercises are self-contained and can be run individually or as part of the complete project structure.
</Note>

### Basic Exercises

The `35-ejercicios-basicos` folder contains foundational exercises that cover:

- Variables and data types
- Conditional statements (`if`, `else`, `elseif`)
- Loops (`for`, `while`, `foreach`)
- Arrays and array manipulation
- String operations
- Mathematical operations and algorithms

**Example from ejercicio-01-08.php:**

```php
$test = 0;

if ($test > 0) {
    echo $test . ' es positivo';
} elseif ($test == 0) {
    echo $test . ' El cero se considera un número neutro';
} else {
    echo $test . ' no es positivo';

```

### Functions

The `ejercicios-funciones` folder includes 10 exercises focusing on:

- Function declaration and parameters
- Return values and arrays
- Prime number detection
- Number classification (even/odd, prime/composite)
- Array operations and transformations

**Example from funcion-01.php:**

```php
function func1() {
    $esPrimo = true;
    $contarPrimos = 0;
    $contarPares = 0;
    $contarImpares = 0;

    for ($i = 0; $i < 100; $i++) {
        // Prime number detection logic
        for ($j = 2; $j < $i; $j++) {
            if ($i % $j == 0) {
                $esPrimo = false;
            }
        }
        
        if ($esPrimo) {
            $contarPrimos++;
        }
        $esPrimo = true;
    }

    return array(
        "PRIMOS" => $contarPrimos,
        "PARES" => $contarPares,
        "NO PARES" => $contarImpares
    );
}
```

### Object-Oriented Programming

The `POO/ejercicio-banco` folder contains a complete banking system simulation:

- **CuentaBanco.class.php**: Base bank account class with deposits, withdrawals, and transactions
- **CuentaAhorro.class.php**: Savings account with interest calculations
- **CuentaCorriente.class.php**: Checking account with overdraft protection
- **Cliente.class.php**: Customer information management
- **Transaccion.class.php**: Transaction history tracking
- **Cajero.class.php**: ATM operations simulation

**Example from CuentaBanco.class.php:**

```php
class CuentaBanco {
    private $numeroCuenta;
    private $tipo;
    private $titular;
    private $saldo;
    private $transacciones = [];

    public function depositar($monto) {
        $this->saldo += $monto;
        $this->RegistrarTransaccion('depósito', $monto);
    }

    public function retirar($monto) {
        if ($monto <= $this->saldo) {
            $this->saldo -= $monto;
            $this->RegistrarTransaccion('retiro', $monto);
        } else {
            throw new Exception("Saldo insuficiente");
        }
    }
}
```

### CRUD Operations

The `CRUD` folder implements a complete user management system:

- **database.php**: PDO database connection with UTF-8 support
- **orm.class.php**: Object-Relational Mapping base class
- **usuario.class.php**: User entity class
- **registroUsuario**: User registration (Create)
- **listarUsuario**: User listing (Read)
- **modificarUsuario**: User updates (Update)
- **borrarUsuario**: User deletion (Delete)

**Example ORM Implementation:**

```php
class ORM {
    protected $id;
    protected $table;
    protected $db;

    // Create
    public function save($data) {
        $columns = array_keys($data);
        $placeholders = array_map(fn($col) => ":$col", $columns);
        
        $sql = "INSERT INTO {$this->table} (";
        $sql .= implode(', ', $columns);
        $sql .= ") VALUES (";
        $sql .= implode(', ', $placeholders);
        $sql .= ")";
        
        $stmt = $this->db->prepare($sql);
        return $stmt->execute($data);
    }

    // Read
    public function getAll() {
        $stmt = $this->db->prepare("SELECT * FROM {$this->table}");
        $stmt->execute();
        return $stmt->fetchAll();
    }

    // Update
    public function updateById($id, $data) {
        $set_clauses = [];
        foreach (array_keys($data) as $key) {
            $set_clauses[] = "{$key} = :{$key}";
        }
        
        $sql = "UPDATE {$this->table} SET ";
        $sql .= implode(', ', $set_clauses);
        $sql .= " WHERE {$this->id} = :id_value";
        
        $stmt = $this->db->prepare($sql);
        return $stmt->execute(array_merge($data, ['id_value' => $id]));
    }

    // Delete
    public function deleteById($id) {
        $stmt = $this->db->prepare("DELETE FROM {$this->table} WHERE {$this->id} = :id");
        $stmt->bindParam(':id', $id);
        return $stmt->execute();
    }
}
```
